Configuration de l'ordre d'amorçage
L'ordre d'amorçage peut être défini sous l'onglet Avancé dans le menu de configuration du BIOS. Les étapes de modification de l'ordre d'amorçage peuvent varier en fonction du modèle de votre ordinateur.
REMARQUE : Les configurations du BIOS varient d'un ordinateur à l'autre. Pour plus d'informations sur un ordinateur spécifique, consultez la documentation fournie avec l'ordinateur ou accédez au document approprié pour votre PC.
PC de bureau HP - Informations sur l'utilitaire de configuration du BIOS et options de menu

Ordinateurs portables HP - Informations de configuration et options des menus du BIOS
Procédez comme suit pour configurer l'ordre d'amorçage sur la plupart des ordinateurs.
Mettez l'ordinateur sous tension ou redémarrez-le.
Lorsque l'écran est vierge, appuyez sur la touche F10 pour entrer dans le menu des paramètres du BIOS.
Sur certains ordinateurs, vous pouvez accéder au menu des paramètres du BIOS en appuyant sur la touche F2 ou F6.
Après avoir ouvert le BIOS, accédez aux paramètres de démarrage.
Pour les PC portables : sélectionnez l'onglet Stockage, puis sélectionnez Options d'amorçage.
Pour les PC de bureau : sélectionnez l'onglet Configuration du système, puis sélectionnez Ordre d'amorçage.
Suivez les indications à l'écran pour modifier l'ordre d'amorçage.
Réinitialisez l'ordre d'amorçage aux paramètres par défaut

Pour réinitialiser l'ordre d'amorçage aux paramètres par défaut, procédez comme suit :
Mettez l'ordinateur sous tension ou redémarrez-le.
Lorsque l'écran est vierge, appuyez sur la touche F10 pour entrer dans le menu des paramètres du BIOS.
Dans l'utilitaire de configuration du BIOS, sélectionnez l'onglet Fichier.
Sélectionnez Appliquer les paramètres par défaut et quitter.
L'utilitaire d'installation du BIOS se ferme, et l'ordinateur démarre Windows.
